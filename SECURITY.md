# Security policy

## Supported version

Only the latest version deployed from the `main` branch is supported.

## Security design

This portfolio is a static website. It has no server-side code, database, authentication, forms, analytics, cookies or secret configuration.

The page applies a restrictive Content Security Policy that permits resources only from the same origin and disables plugins, framing, form submission and mixed HTTP content.

## Reporting a concern

Please report security concerns privately by email to `ahmad.mahmoud@tum.de`. Do not include credentials, private keys or sensitive personal data in a public GitHub issue.
